## What is KerfOs?

KerfOs is an AI-powered cabinet design tool that makes professional fabrication accessible to DIYers and small shops.

### Features
- Cabinet Builder UI with drag-and-drop components
- Real-time 3D visualization
- Cut list generation and optimization
- Hardware sourcing integration
- Material cost calculation
- PDF/DXF/CSV export

### Technology Stack
- **Frontend:** Next.js 14, React, TypeScript, Three.js
- **Backend:** FastAPI, PostgreSQL, SQLAlchemy
- **Authentication:** JWT tokens
- **Billing:** Stripe subscriptions
- **Deployment:** Railway (backend) + Vercel (frontend)
